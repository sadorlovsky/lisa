## Installation
```bash
npm install -g lissie
```

## Usage
### CLI
```bash
$ lissie mit -a "Zach Orlovsky"
```

### Node.js
```javascript
import lissie from 'lissie'

lissie({
  license: 'mit'
  author: 'Zach Orlovsky',
  email: 'sadorlovsky@gmail.com'
})
```
